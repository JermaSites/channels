### Updating

This list updates once per day with the latest stats for each channel. Full details will be available in a soon-to-be-published article.

### Adding channel

To add a channel, edit [`channels.txt`](https://github.com/JakeSteam/Jerma/blob/main/automation/channels.txt) to include the channel ID, channel name, and a ðŸ¶ if it's a truly impressive channel:
* Channel ID is available at `About` -> `Share` -> `Copy channel ID`
