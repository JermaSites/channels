## How it works

### Updating

This list updates once per day with the latest stats for each channel. Full details [are available here](https://blog.jakelee.co.uk/fetching-youtube-metadata-in-github-actions-and-persisting/).

### Adding channel

To add a channel, edit [`channels.txt`](https://github.com/JakeSteam/Jerma/blob/main/automation/channels.txt) to include the channel ID, channel name, and a ðŸ¶ if it's a truly impressive channel:
* Channel ID can be retrieved from a profile at `About` -> `Share` -> `Copy channel ID`
* Channel name is not used, except to make the file more readable.
